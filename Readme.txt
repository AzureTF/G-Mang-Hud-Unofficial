G-Mang Unofficial Beta 32

To install:

1. Unzip or extract "G-Mang-Hud-Unofficial.zip"

2. Navigate to your tf directory ( C:\Program Files (x86)\Steam\steamapps\common\Team Fortress 2\tf )

2. Open the "custom" folder.

3. Move the folder "G-Mang-Hud-Unofficial" that you extracted or unzipped in step 1 into the "custom" folder
